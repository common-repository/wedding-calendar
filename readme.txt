== Description ==
Wedding Calendar allows the website owner to create a custom calendar of with event dates marked. An example of that would be Wedding dates. Its enter the dates which you want to mark in red in the plugin & put the short code in the section where you wish to display the calendar.

== Frequently Asked Questions ==

= Why use this plugin work? =
If you need to add a custom calendar on your website, this is the plugin for you. This plugin allows you to mark certain dates in a calendar & show it any where on your website with the help of a shortcode.


= How does this plugin work? =
1. Just add weding calender to your website.

2. Find the wedding calendar settings from the left menu options.
enter the dates you wish to mark in red for your calendar, in the format entioned there.

3. Copy the shortcode & paste it anywhere you wish the calendar to be displayed.

== Screenshots ==

1. Find Wedding Calendar in the menu.
2. Insert your dates in the given format.
3. Copy & paste the short code where you wish to display the calendar.


== Installation ==
1. Upload "wedding-calendar.php" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
